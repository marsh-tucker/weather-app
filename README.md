# Weather App 🌤️

This is a full-stack weather forecast web app built in C# using ASP.NET Core MVC. It allows users to input a ZIP code and get real-time weather data using the OpenWeather API.

## Features
- ASP.NET MVC architecture
- Razor views for UI
- Async API calls to OpenWeather
- Input validation
- (Coming soon) Database for search history tracking
